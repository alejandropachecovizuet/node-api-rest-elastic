Api Rest - Mesa

General:20000

  • PUT /:index/:type/:id
    • Desc:Agrega un registro en el indice :index, el tipo :type y el id :id
    • Header:x-access-token=Token
    • Header:ux=Usuario con el que se creó el Token
    • JSON-Example: {"cve": "1","description":"Activo"}
  • POST /:index/_search
    • Desc:Busca todos los registros en el indice :index que cumplan con el query
    • Header:x-access-token=Token
    • Header:ux=Usuario con el que se creó el Token
    • JSON-Example:{"query": {"match_all": {}}}
  • POST /:index/:type/_search
    • Desc:Busca todos los registros en el indice :index y tipo :type que cumplan con el query
    • Header:x-access-token=Token
    • Header:ux=Usuario con el que se creó el Token
    • JSON-Example:{"query": {"match_all": {}}}
  • GET /:index/_searchById/:id
    • Desc:Busca el registro :id en el indice :index
    • Header:x-access-token=Token
    • Header:ux=Usuario con el que se creó el Token
  • PUT /:index/:type/:id/_update
    • Desc:Actualiza el registro indicado en :id(_id de elasticsearch) del catalogo :index y tipo :type
    • Header:x-access-token=Token
    • Header:ux=Usuario con el que se creó el Token
    • JSON-Example:NO_BODY
  • DELETE /:index/:type/:id
    • Desc:Borra el registro indicado en :id(_id de elasticsearch) del catalogo :index
    • Header:x-access-token=Token
    • Header:ux=Usuario con el que se creó el Token
    • JSON-Example:NO_BODY
  • GET /changes
    • Desc:Regresa la fecha última en que fuerón actualizados los catalogos

Login:20001

  • POST /authenticate
    • Desc:Authenticación del usuario
    • JSON-Example:{"username":"luisXV", "pwd":"123456"}
    • Response:Usuario + Token

Loopback:20002

  • POST /loopback
    • Desc:Servicio loopback de usuario
    • Esteban:prueba
    • JSON-Example:{"username":"luisXV", "nombre":"Luis", "apellidop":"Zamora", "apellidom":"Salgado", "frase":"123456", "loopback":{"http":{"code":"404","response":{"Err":"ok","x":"y"}} } }
